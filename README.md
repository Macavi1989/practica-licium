# practica-licium
Un módulo de backend para Licium que permite crear y gestionar listas de tareas internas. Permite añadir, editar y borrar tareas, marcar tareas como completadas, cerrar o reabrir listas, gestionarlo desde el panel de administración y controlar quién puede acceder usando permisos y grupos. Es útil para aprender paso a paso cómo crear módulos.
